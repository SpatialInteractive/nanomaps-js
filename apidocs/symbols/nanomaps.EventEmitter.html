advise(methodName, advice, target)
Overrides the given methodName on this instance applying the given advice ("before" or "after"). The given target method is invoked for the advice.

Invoking this method supports the arbitrary augmentation of a class's methods with before or after advice. The original method is replaced with a stub that invokes a list of other methods before and after the original target. If the original method doesn't exist, the stub will still be defined.

It is safe to use this method on prototypes or instances as it will only modify the outermost prototype (target of the call). 

MapSurface.prototype.advise('initialize', 'after', 
		function() { alert('Initialized'); });
myMap.advise('onzoom', 'after', 
		function() { alert('zoomed'); });
Parameters:
{string} methodName
Name of the method to override
{string} advice
Advice to apply ("before" or "after")
{function} target
Function which is the target of advice
emit(event)
Invokes a given event by name, triggering all registered persistent listeners and once only listeners. All arguments after the first are taken to be the arguments to the event listeners. Once only listeners are invoked first followed by persistent listeners. Otherwise, listeners are invoked in the order added.

If a method exists on the instance named 'on' + event, then that method will be invoked as an event listener prior to any others.
